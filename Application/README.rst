Application
=========================

This is an application scaffold for Molo_.

Getting started
---------------

To get started::

    $ virtualenv ve
    $ pip install -e .
    $ ./manage.py migrate
    $ ./manage.py createsuperuser
    $ ./manage.py runserver

You can now connect access the demo site on http://localhost:8000


.. _Molo: https://molo.readthedocs.org
